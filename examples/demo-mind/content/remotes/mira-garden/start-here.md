# Mira's public garden

> You are reading a **mental node** — a read-only mirror of *someone else's*
> Atlas Mind, subscribed to from this one.

This little cluster does not live in this mind. It is published by another Atlas
instance (a fictional "Mira") as a **node**, and this demo simply *follows* it.
You can read it, search it, and link to it — but you can't edit it: notice the
teal accent, the **Mental nodes** label in the sidebar, and the missing edit
button up top.

That is federation: minds that point at **other minds**, in the open, without a
central server in the middle. Your knowledge can cite a teammate's node the same
way one note cites another.

## How it connects back

Open the graph view (the constellation icon, top of the sidebar, or `Ctrl/Cmd+G`)
and you'll see this node as its own **teal island**, wired into the main mind
through a few links. That single picture is the whole pitch: not a folder, a
*network of minds*.
