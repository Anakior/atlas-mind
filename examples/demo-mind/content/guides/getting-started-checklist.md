# Getting-started checklist

A few things to try in this demo — and to do when you run your **own** Atlas Mind.
Every checkbox below also shows up in the **Tasks** view (the checklist icon at
the top of the sidebar), gathered together with the tasks from every other
document in the mind.

> In this **offline demo** the checkboxes are read-only. In the live engine,
> ticking a box writes the change straight back to the file.

## Explore this demo

- [x] Open a document and read it
- [x] Run a search — try `wikilink` or `notion`
- [ ] Open **The Mind** graph with `Ctrl/Cmd+G`
- [ ] Find a highlighted passage with a margin note
- [ ] Open the **Tasks** view and click a task to jump straight to it

## Run your own mind

- [ ] `pip install atlas-mind`, then `atlas init my-mind`
- [ ] Keep it in a git repo **you** own
- [ ] Connect it to your AI over MCP
- [ ] Share a slice with a friend — the [[guides/hive-mind|Hive Mind]]
